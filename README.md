# AudioVisualizer

A lightweight C++ audio visualization project using modern OpenGL.

## Overview

This project renders real-time audio visualizations using C++, OpenGL, and supporting libraries.  
The goal is to provide a simple, modular foundation for experimenting with graphics, audio processing, and shader-based visual effects.

## Tools and Technologies

- C++20  
- CMake  
- OpenGL   
- GLFW (window and input handling)  
- GLAD (OpenGL function loading)  
- Conan (dependency management)
